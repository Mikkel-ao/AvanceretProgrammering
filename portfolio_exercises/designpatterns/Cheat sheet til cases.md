# ✅ Facit: Design Pattern Cheat Sheet til Cases

| Case nr. | Problem                                         | Pattern            |
|----------|--------------------------------------------------|--------------------|
| 1        | Logge til flere destinationsformer               | **Decorator**      |
| 2        | Knapper udløser handlinger, evt. fortryd         | **Command**        |
| 3        | Gammelt interface skal passe med nyt             | **Adapter**        |
| 4        | Ét fælles objekt til deling af tilstand          | **Singleton**      |
| 5        | Bygge dokument med fleksibel struktur            | **Builder**        |
| 6        | Automatisk besked til mange ved ændring          | **Observer**       |
| 7        | Forskellige brugertyper med fælles metode        | **Factory Method** |
| 8        | Filer og mapper behandles ens                    | **Composite**      |
| 9        | Tung service instansieres kun ved behov          | **Proxy**          |
| 10       | Skiftbar sortering af data                       | **Strategy**       |
